Основы HTML

HTML - язык разметки гипертекста - предназначен для создания Web-страниц. HTML представляет собой достаточно простой набор кодов, которые описывают структуру документа. HTML позволяет выделить в тексте отдельные логические части (заголовки, абзацы, списки и т.д.), поместить на Web-страницу подготовленную фотографию или картинку, организовать на странице ссылки для связи с другими документами.

Основные компоненты HTML:
  • Тег. Тег HTML это компонент, который командует Web- броузеру выполнить определенную задачу.
  • Атрибут . Атрибут HTML изменяет тег.
  • Значение. Значения присваиваются атрибутам и определяют вносимые изменения.

HTML- программа должна начинаться тегом <HTML> и заканчиваться тегом </HTML>


       <HTML>
               ......    (здесь будут другие теги) 
       </HTML>

HTML включает в себя два главных тега: заголовок <HEAD> и </HEAD> и тело <BODY> и </BODY> 


                   <HTML>
                     <HEAD>
                           ......    (здесь будет заголовок)
                     </HEAD>

                     <BODY>
                     ......    (здесь будут другие теги тела)
                     </BODY>
                     
                   </HTML>
