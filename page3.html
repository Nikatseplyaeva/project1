Основы CSS

CSS — это отдельный язык со своим форматом и правилами, но он создан специально для HTML. С помощью CSS можно описать внешний вид страницы и отдельных элементов. Например, цвет текста, размер шрифта, размеры и отступы картинок, рамки вокруг элементов, закругления в фотках и так далее.

В CSS используются атрибуты. Они вставляются в тег, изменяя свойства текста, абзаца и тд.

Пример (Здесь мы используем атрибут background для создания фонового изображения, а также его цвета):


       <style>
       <body>  {
               background: #000 url(nubex.png); /* Фоновый цвет и фоновый рисунок*/
               color: #fff; /*Цвет текста на странице*/ 
               }    
       </style>
       </body>
